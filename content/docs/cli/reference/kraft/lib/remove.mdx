---
title: "kraft lib remove"
description: Removes a library dependency from the project directory
---

Remove a Unikraft library from the project directory.


```
kraft lib remove [FLAGS] LIB
```

### Examples

```
# Remove a library from the project
$ kraft lib remove libfoo

```

## Options

```
  -h, --help               help for remove
  -K, --kraftfile string   Set an alternative path of the Kraftfile
  -w, --workdir string     workdir to remove lib from
```
