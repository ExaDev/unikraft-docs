List compose projects.

```
kraft compose ls [FLAGS]
```

### Examples

```
# List all compose projects
$ kraft compose ls

```

## Options

```
  -a, --all             Show all projects (default shows just running)
  -h, --help            help for ls
  -o, --output string   Set output format (default "table")
```

## Options inherited from parent commands

```
  -f, --file string   Set the Compose file.
```
