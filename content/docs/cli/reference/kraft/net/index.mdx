---
title: "kraft net"
description: Manage machine networks
---

Manage machine networks.

```
kraft net SUBCOMMAND
```

### Examples

```
# Create a new network
$ kraft network create my-network

```

## Options

```
  -d, --driver string   Set the network driver. (default "bridge")
  -h, --help            help for net
```
