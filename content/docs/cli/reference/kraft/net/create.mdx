Create a new machine network.

```
kraft net create [FLAGS] NETWORK
```

### Examples

```
# Create a new machine network
$ kraft network create my-network --network 133.37.0.1/12

```

## Options

```
  -h, --help             help for create
  -n, --network string   Set the gateway IP address and the subnet of the network in CIDR format.
```

## Options inherited from parent commands

```
  -d, --driver string   Set the network driver. (default "bridge")
```
