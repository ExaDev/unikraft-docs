title: "kraft remove"

---

Remove one or more running unikernels


```
kraft remove [FLAGS] MACHINE [MACHINE [...]]
```

### Examples

```
# Remove a running unikernel
$ kraft rm my-machine

```

## Options

```
      --all           Remove all machines
  -h, --help          help for remove
  -p, --plat string   Set the platform virtual machine monitor driver.  Set to 'auto' to detect the guest's platform and 'host' to use the host platform. (default "auto")
```
